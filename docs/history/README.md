# History Archive

This folder stores historical project notes that are useful for context but are not part of day-to-day navigation.

## What belongs here

- Expansion summaries
- Migration/reorganization notes
- One-time rollout or phase completion documents
- Superseded index snapshots

## What should not be added here

- Current operational runbooks
- Active reference guides
- Canonical navigation files

Use [../index.md](../index.md) for current handbook navigation.
